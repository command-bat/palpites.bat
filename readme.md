```md
# palpites.bat

Plataforma de palpites e previsões com integração entre frontend e backend.

## ✨ Visão geral

Aplicação full-stack voltada para criação e gerenciamento de palpites, com API própria e interface dinâmica.

## 🚀 Tecnologias

- Next.js
- React
- HTML
- CSS
- Node.js
- Express
- API REST

## 📦 Funcionalidades

- Cadastro e gerenciamento de palpites
- Consumo de API REST
- Interface dinâmica
- Estrutura full-stack

## ⚙️ Como rodar o projeto
```
```bash
npm install
npm run dev
```
